# 🚀 2D Space Shooter Game

A classic arcade-style **2D Space Shooter** built using **Python** and **Pygame**. Control your spaceship, destroy enemy ships, survive as long as possible, and beat your highest score.

---

## 🎮 Features

- 🚀 Smooth player movement
- 🔫 Shooting mechanics
- 👾 Random enemy spawning
- 💥 Explosion animation
- ❤️ Health system
- 🏆 High score saving
- ⏸ Pause menu
- ⚙ Settings menu
- 🎵 Background music & sound effects
- 🔊 Adjustable volume
- ⌨ Custom shoot key binding
- 🖥 Responsive game window
- 🎨 Modern UI with hover effects

---

## 🛠 Built With

- Python 3
- Pygame

## 🎯 Controls

| Key | Action |
|------|--------|
| ← → ↑ ↓ | Move spaceship |
| Space | Shoot (Default) |
| P | Pause / Resume |
| Mouse | Navigate menus |

> The shooting key can be changed from the Settings menu.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/chiragpoddar007/2D-Space-Shooter.git
```

Move into the project folder

```bash
cd 2D-Space-Shooter
```

Install Pygame

```bash
pip install pygame
```

Run the game

```bash
python main.py
```

---

## 📖 How to Play

1. Start the game.
2. Move your spaceship.
3. Shoot incoming enemies.
4. Avoid collisions.
5. Try to survive and achieve the highest score.

---

## 📊 Game Mechanics

- Enemy speed increases over time.
- Enemy spawn delay gradually decreases.
- Health decreases when enemies collide with the player.
- The highest score is automatically saved.
- Explosion effects are displayed after destroying enemies.
